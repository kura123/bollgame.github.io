# bollgame
簡単なボールゲームアプリ

<img width="175" alt="bollgame github io_img" src="https://user-images.githubusercontent.com/28621401/54908560-88dec680-4f2b-11e9-9018-70dbe8519b14.PNG">


## Description
- 加速度センサーを使った玉転がしゲーム
- スマホを傾けてボールを転がす
- 赤い四角までボールを運べばクリア
